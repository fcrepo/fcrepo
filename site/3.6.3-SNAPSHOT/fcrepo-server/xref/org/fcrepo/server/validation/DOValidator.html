31      /**
32       * Validates a digital object.
33       * 
34       * @param in
35       *        The digital object provided as a bytestream.
36       * @param validationLevel
37       *        The level of validation to perform on the digital object. This is
38       *        an integer from 0-2 with the following meanings: 0 = VALIDATE_ALL
39       *        (do all validation levels) 1 = VALIDATE_XML_SCHEMA (perform only
40       *        XML Schema validation) 2 = VALIDATE_SCHEMATRON (perform only
41       *        Schematron Rules validation) -1 = VALIDATE_NONE (no validation)
42       * @param phase
43       *        The stage in the work flow for which the validation should be
44       *        contextualized. "ingest" = the object is in the submission format
45       *        for the ingest stage phase "store" = the object is in the
46       *        authoritative format for the final storage phase
47       * @throws ServerException
48       *         If validation fails for any reason.
49       */
50      public void validate(InputStream in,
51                           String format,
52                           int validationLevel,
53                           String phase) throws ServerException;
