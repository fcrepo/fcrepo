Project Modules

This project has declared the following modules:

Name Description
Fedora Repository Common Module Common, generally useful utilities
Fedora Repository Kernel API Fedora Repository Kernel API, containing types and constants for working with Fedora
Fedora Repository Kernel Implementation Fedora Repository Kernel Implementation, containing implementation of core kernel API interfacess
Fedora Repository Deployable Web Application The Fedora web application
Fedora Event Serialization A module for serializing Fedora events as RDF
Fedora Repository JMS Module The Fedora Commons repository JMS module: Provides repository event publication via Java Message Service.
Fedora Repository HTTP Commons Module The Fedora Commons repository HTTP commons module: Provides HTTP (JAX-RS) resources and helpers for use in assembling HTTP APIs over the Fedora Commons repository framework.
Fedora Repository HTTP API Module The Fedora Commons repository HTTP API: Provides a RESTful HTTP API to interact with the Fedora Commons repository.
Fedora Repository Authorization Commons Module Provides unified policy enforcement points for various access control implementations.
Fedora Repository Configurations Module The Fedora Commons repository configurations module: Provides configuration resources that are used in integration testing and deployments.
Fedora Commons :: Parent POM Parent POM for Fedora Commons Projects
Fedora Repository LDP integration tests Module Packaging the LDP integration test suite for fcrepo4
Fedora Repository RDF integration tests Module Fedora RDF integration scenarios
Fedora Repository WebAC Authorization Module An authorization module for the Fedora Commons Repository framework, implementing the RDF-based WebAccessControl authorization mechanism, as proposed by the W3C. This makes it possible to define fine-grained repository access controls on individual resources or classes of resources to specific users and/or groups.
Fedora Repository Persistence API Module Provides a common interface for persistence operations
Fedora Repository Persistence Common Module Provides implementations for common persistence operations
Fedora Repository OCFL Persistence Impl Fedora Repository OCFL Persistence Implementation, containing implementation of core persistence API interfaces
Fedora Repository Search API Fedora Repository Search API interfaces and classes
Fedora Repository Search Impl Fedora Repository Default Search API Implementation
